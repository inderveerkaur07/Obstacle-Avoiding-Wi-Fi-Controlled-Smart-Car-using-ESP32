# 🚗 Obstacle Avoiding Wi-Fi Controlled Smart Car using ESP32

## 📌 Project Overview

This project is an IoT-based smart robotic car that can be controlled wirelessly through Wi-Fi using a mobile device. The car is built using an ESP32 microcontroller and includes an ultrasonic sensor for automatic obstacle detection and collision avoidance.

The project demonstrates the practical implementation of IoT, embedded systems, wireless communication, and robotics.

## ✨ Features

- 📶 Wi-Fi Based Remote Control
- 🤖 ESP32 Web Server Control Interface
- 🎮 Joystick-Based Control System
- 📡 Ultrasonic Obstacle Detection
- 🚧 Automatic Collision Avoidance
- 🔄 Forward, Backward, Left & Right Movement
- 📱 Mobile Browser Control
- ⚡ Real-Time Response

## 🛠 Components Used

- ESP32 Development Board
- HC-SR04 Ultrasonic Sensor
- L298N Motor Driver Module
- DC Motors
- Robot Chassis
- Wheels
- Battery Pack
- Jumper Wires

## ⚙️ Working Principle

1. ESP32 creates its own Wi-Fi hotspot.
2. User connects a mobile device to the hotspot.
3. A web-based control page opens in the browser.
4. Commands are sent through a joystick interface.
5. ESP32 controls the motors through the L298N motor driver.
6. The ultrasonic sensor continuously monitors obstacles.
7. If an obstacle is detected within the safety range, the car automatically stops or reverses.
